KanjiPairs is a Single HTML page Kanji Matching game. It was created with the goal in mind to make a small game in JavaScript that can be run by just grabbing the files and opening the base HTML file in a modern web browser. Of course, it can also be hosted on a web server for users to access w/o having to have it downloaded locally.

The game is a matching game where you pick two kanji and try to match on the reading given on the back of the card. A kanji may have many readings, but for the sake of the matching game, only one reading will be valid for matching against, the one shown on the back of the card.

This game does not communicate with or store any personal data on a server. If it's downloaded from a server, that's the only server interaction that happens. Filters and game settings are stored completely locally on the client (your web browser) via HTML 5 Local Storage.

The library jQuery UI Touch Punch is used to enable touch events for the slider on mobile.
